# Cognitive Amplification vs Cognitive Delegation in Human-AI Systems

This repository accompanies the paper:

**Cognitive Amplification vs Cognitive Delegation in Human-AI Systems: A Metric Framework**

## What this project is about

This project introduces a metric framework for distinguishing between:

- **cognitive amplification**, where AI improves hybrid performance while preserving human capability
- **cognitive delegation**, where AI use progressively displaces human cognitive contribution and risks long-term deskilling

The framework is organized around four quantities:

- **CAI\*** — Cognitive Amplification Index
- **D** — Dependency Ratio
- **HRI** — Human Reliance Index
- **HCDR** — Human Cognitive Drift Rate

Together, these quantities make it possible to evaluate not only short-term hybrid performance, but also whether human capability is being preserved over time.

## Repository contents

This repository contains the NetLogo simulation and associated materials used to explore the framework under controlled conditions.

## Main result

The experiments distinguish three practically relevant regimes:

- degenerate AI-dominated delegation
- human-preserving but weakly competitive interaction
- intermediate boundary regimes that approach the AI baseline while remaining structurally dependent

A central result is that **strong hybrid performance does not necessarily imply genuine amplification**.
